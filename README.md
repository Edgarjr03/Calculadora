# Calculadora / Calculator

## Español
Aplicación de escritorio con interfaz gráfica desarrollada en C# con Visual Studio.
Realiza operaciones matemáticas básicas como suma, resta, multiplicación y división.

### Tecnologías
- C#
- Windows Forms
- Visual Studio 2022

### Cómo ejecutar
1. Clona el repositorio
2. Abre el archivo `.sln` en Visual Studio
3. Presiona `F5` para ejecutar

---

## English
Desktop application with graphical interface developed in C# with Visual Studio.
Performs basic mathematical operations such as addition, subtraction, multiplication, and division.

### Technologies
- C#
- Windows Forms
- Visual Studio 2022

### How to run
1. Clone the repository
2. Open the `.sln` file in Visual Studio
3. Press `F5` to run
